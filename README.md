# How To Install

#### - composer update

#### - npm update

#### - npm run dev

#### - php artisan key:generate

#### - create .env

#### - in .env : DB_CONNECTION=sqlite, BROADCAST_DRIVER=pusher

#### - Setting PUSHER_APP_ID, PUSHER_APP_KEY, PUSHER_APP_SECRET, PUSHER_APP_CLUSTER

#### - php artisan serve
